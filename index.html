📖 How to Use Spatial Audio Stage

🎧 Spatial View

  • Click on the canvas to place a sound node
  • Drag a node to move it in 3D space
  • Shift+Click to multi-select nodes
  • Double-click a node to open the inspector
  • Mouse wheel to zoom in/out
  • Right-click drag to pan the canvas
  • Delete key to remove selected nodes

🎬 Recording Movement

  • Click Rec Move (or press R) — 3s countdown starts
  • Drag any node freely during recording — no selection needed
  • Other already-recorded nodes will animate alongside
  • Click Rec Move again or press Pause to stop recording
  • Re-recording a node with existing keyframes only discards keyframes after the start time

📼 Timeline View

  • Click on a track lane to create a clip
  • Drag a clip to move it in time
  • Drag edges of a clip to trim start/end
  • Drag items from the Sound Library to lanes
  • Double-click a track name to rename it
  • M / S buttons to mute or solo a track
  • ✂ Split button or ⌘K — split clip at playhead
  • Fade handles — drag translucent handles on clips to set fade in/out
  • Clip fades — adjust fade in/out duration in inspector

🎵 Sound Library & Folders

  • Folders — organize sounds into custom folders via the dropdown
  • Use ＋ to create a folder, ✕ to delete (items move to Uncategorized)
  • Click a folder to filter the sound list
  • Upload audio via File menu — duplicates are detected and merged
  • Cached audio survives page refresh (IndexedDB)

🎚 EQ (Equalizer)

  • Main EQ — 3-band EQ per node: Low (250Hz), Mid (variable), High (4kHz)
  • Clip EQ — select a clip in timeline to set per-clip EQ that overrides the main EQ
  • Adjust sliders in the inspector — applied in real-time during playback
  • Each clip can have independent EQ settings

🔗 Grouping

  • Select 2+ nodes, click Group
  • Same Entity — all nodes move to the same spot
  • Different Entities — maintain relative offsets while moving
  • Ungroup to break the link

⌨️ Keyboard Shortcuts

  • Space — Play / Pause
  • R — Record movement
  • Delete — Remove selected
  • Cmd+Z — Undo
  • Cmd+Shift+Z — Redo
  • ⌘K — Split clip at playhead
  • ⌘C — Copy selected clip
  • ⌘V — Paste clip at playhead
